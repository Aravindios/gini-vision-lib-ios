GiniVisionDocumentBuilder

public class GiniVisionDocumentBuilder
  • The GiniVisionDocumentBuilder provides a way to build a GiniVisionDocument from a Data object. The DocumentSource must be provided in the initialization, being optional but highly recommended setting the DocumentImportMethod afterwards. This could be an example of how a GiniVisionDocument should be built when it has been imported with the Open with feature.
let documentBuilder = GiniVisionDocumentBuilder(data: data, documentSource: .appName(name: sourceApplication))
documentBuilder.importMethod = .openWith
let document = documentBuilder.build()
